# CCPDate(swift5.0, iOS10.0+, v1.0.0)
Date常用的扩展
# How to use
1. pod 'CCPDate'
2. import CCPDate
3. See demo
## Demo
* DateDescription date转字符串.（中文，英文，星期中英文...）
* DateComponents date相关的属性.(year, month, day...)
* DateCalculation date日期比较.(add,加一段时间(year, month..., combine)后的日期；sub, 两个日期相减后大致的时间差, 准确时间差)
